## Project setup

To deploy this project locally please follow these steps:

### 1. Add Host
```
sudo vim /etc/hosts

127.0.0.1      demo.juicyllama
::1            demo.juicyllama
```

### 2. Install & Run

```
npm install
```
```
npm run serve
```
